# Website Starter

간단한 정적 웹사이트 시작 파일입니다.

## 파일

- `index.html`: 페이지 구조
- `styles.css`: 색상과 레이아웃

## GitHub 연결

1. GitHub에서 새 저장소를 만듭니다.
2. 아래 명령으로 원격 저장소를 연결합니다.

```bash
git remote add origin <YOUR_GITHUB_REPO_URL>
git add .
git commit -m "Initial website"
git push -u origin main
```

## 로컬에서 보기

브라우저에서 `index.html`을 열면 됩니다.
