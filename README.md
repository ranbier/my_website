# Website Starter

정적 포트폴리오 사이트와 문서 원본이 함께 들어 있습니다.

## 주요 페이지

- `index.html`: 메인 허브 페이지
- `systems.html`: 시스템 구조
- `navigation.html`: 경로/제어
- `perception.html`: 센서 융합
- `infrastructure.html`: ROS2, Docker, 네트워크
- `troubleshooting.html`: 문제 해결 패턴
- `insights.html`: 반복되는 인사이트
- `philosophy.html`: 엔지니어링 철학

## 원본 문서

- `robotics-knowledge-base/`: Markdown 기반 원본 문서

## 배포

GitHub와 Cloudflare Pages에 연결해 정적 사이트로 배포할 수 있습니다.
