# Autonomous Robotics Engineering

자율주행 및 수중 로봇 시스템을 설계하고 구현하면서 축적한 기술 문서.

## 핵심 영역

- Navigation (경로 생성 및 추종)
- Sensor Fusion (IMU, DVL, GPS)
- Control (PID, MPC)
- ROS2 기반 시스템 아키텍처
- 실제 문제 해결 경험

## 특징

- 이론이 아니라 실제 구현 중심
- 재사용 가능한 구조 설계
- 실패와 문제 해결 과정 포함

## 주요 프로젝트

- Waypoint System
- AUV Navigation Stack
- Sensor Fusion Pipeline
